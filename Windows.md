# Windows

## Package Managers

### Winget

- Official Microsoft package manager
- Search packages: <https://winget.run/>

```shell
winget upgrade --all
```

### Scoop

<https://scoop.sh/>

```shell
scoop update *
```

### Chocolatey

<https://chocolatey.org/>

```shell
choco upgrade all
```
